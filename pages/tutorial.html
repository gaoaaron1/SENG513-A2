Game's Objective:

Costume Warriors is a two-player game

players use keyboard keys to control their character.

The objective is to kill each other before time runs out.

How To Play:

  • Begin in a landscape map with a timer and player healthbar.
  • Player 1 uses 'A' key to move left, 'D' key to move right

    • 'W' key to jump and 'S' key to attack.

  • Player 2 uses 'Left' key to move left, 'Right' key to move right

    • 'Up' key to jump and 'Down' key to attack.

  • If Player 1 strikes and collides with Player 2

    Player 2 loses health and vice versa.

  • The game ends when one of the Player's health reaches 0

    • and the opposing player wins.

  • If time runs out before either player's health reaches 0

    • the player with more health wins.

  • If time runs out before either player's health reaches 0

    • and both players have same health, end in a draw.

Game Mechanics:

Players try to attack one another and the one whose health reaches 0 first wins.

If time runs out the winner is the one with more health.

If time runs out and both players have same health, then end game in draw.
